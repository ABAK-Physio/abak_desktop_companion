# ABAK Desktop Companion

ABAK Desktop Companion est une application de bureau gratuite destinée à compléter l'application mobile ABAK.

Elle permet de centraliser les résultats des exercices réalisés sur mobile, de les associer à des patients et de préparer leur exploitation dans un contexte clinique.

Le projet est développé avec Flutter et est disponible pour **macOS** et **Windows**.

---

# Fonctionnalités

- import des fichiers `.abak` générés par ABAK ;
- association des résultats aux patients ;
- gestion des épisodes cliniques ;
- consultation de l'historique des exercices ;
- génération de rapports cliniques ;
- lecture de la Carte Vitale (fonctionnalité en cours d'intégration).
